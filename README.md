Personal Finance & Expense Intelligence Dashboard

Built an Excel-based dashboard to track income, expenses, savings, and budget performance.

Features:
- KPI metrics (Income, Expense, Savings %)
- Category-wise spending analysis
- Monthly expense trend
- Budget vs Actual comparison
- Overspending alert system

Tools Used:
- Microsoft Excel
- Pivot Tables
- SUMIFS, IF formulas
- Data Cleaning

Insights:
- Shopping and Bills are major expense drivers
- Expenses show increasing trend over months
- Most categories exceed planned budget
